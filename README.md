# MinerJob
Simple Miner and ore delivery job script for SA-MP
Locatoed at query
Author: Palwa

# Installation
* Download / Copy Miner.pwn code
* Compile the .pwn version
* Run / Restart your server
* Enjoy!

# InGame Command
* /mineore
* /loadore
* /takeore
* /unloadore
